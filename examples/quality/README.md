# Quality Examples

Examples for `color.quality`.

## Run

```powershell
.\.venv\Scripts\python.exe examples\quality\example_01_ssi.py
```

## Examples

| File | Demonstrates |
| --- | --- |
| `example_01_ssi.py` | Academy SSI for D65, generated LED, Gaussian and blackbody spectra |

The example prints rounded and unrounded SSI scores. SSI compares spectral
similarity only; it is not a colour rendering index.
