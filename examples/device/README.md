# Device Examples

Examples for display-primary response optimisation.

Run from the repository root:

```powershell
.\.venv\Scripts\python.exe examples\device\example_01_melanopic_silent_substitution.py
```

## Examples

| Example | Demonstrates |
| --- | --- |
| `example_01_melanopic_silent_substitution.py` | Synthetic RGBC LMS-silent and XYZ-silent melanopic endpoints, held-response residuals, and batch melanopic range |

Generated figures are written to `examples/device/output/`.
