Paper spectra

The reflectance spectra of coloured newsprint, paper and cardboard samples. The
samples were collected from various sources and there should be quite covering
set of different colors. There are 18 samples for newsprint, 72 samples for paper
and 70 samples for cardboard, The samples were measured both specular component
included and excluded. When the specular component was included the spectrofotometer
was calibrated with the surface mirror and when specular component was excluded the
spectrofotometer was calibrated by using the white plastic chip provided by the
spectrofotometer manufacturer. The spectra consist of measurements from single
sheet of paper with black and white background and opaque pile of paper sample.

Measuring equipment: Minolta CM 2002 spectrofotometer.

Wavelength interval: 400 nm - 700 nm.

Wavelength resolution: 10 nm.

File formats:  The available formats are ASCII and MATLAB.

The ASCII color spectra are stored in the file paper400_700_10.tar.gz which
consists of files named as newsprintsci.asc, newsprintsce.asc, papersci.asc,
papersce.asc, cardboardsci.asc, cardboardsce.asc and mirrorsci.asc. So the
filenames tell which kind of samples are measured and the endings sci and sce
means whether the specular component is included or excluded in the measured
data. The file mirrorsci.asc consists of the calibration spectra measured from
surface mirror. Every line in each ASCII file contains one spectrum which is
31 component vector.

The file paper_matlab.tar.gz includes MATLAB files which names are the same
as the ASCII files except the file extension ".asc" is replaced by the extension 
".mat". Each file consists of one matrix and the spectra are stored in column
order in the matrices.
